## Project Overview
This project presents an AI-based smart indoor environment monitoring and control system using ESP32-S3 and TinyML.

The system:
- Monitors temperature, humidity, air quality, CO2, occupancy, and lighting
- Uses a Random Forest TinyML model
- Performs real-time edge inference
- Integrates with Blynk IoT cloud
- Controls relays, LEDs, buzzer, and LCD automatically
